  <behavior>
    should_synthesize(text: str) -> bool:
    - Returns True if sum(1 for c in text if c.isalpha()) >= 3
    - Same logic as translator._is_symbol_only but inverted and applied to ru_text

    synthesize_all(segments, voice_id, output_dir, resume=False) -> List[SubtitleSegment]:
    - Creates os.makedirs(os.path.join(output_dir, "segments"), exist_ok=True) at start
    - Builds ElevenLabs client: ElevenLabs(api_key=settings.elevenlabs_api_key)
    - Checkpoint path: os.path.join(output_dir, f"{video_id}_checkpoint.json") — BUT
      synthesize_all does not have video_id parameter. Use a JobState parameter OR
      accept checkpoint_path as parameter OR derive from output_dir. The stub signature is:
        synthesize_all(segments, voice_id, output_dir, resume=False)
      Since no job/checkpoint_path param exists in stub, checkpoint.save() inside synthesize_all
      needs a path. Resolution: accept an optional job: JobState | None = None parameter and
      an optional checkpoint_path: str | None = None parameter added after resume. When both
      are None, skip checkpointing (backward compatible). Tests will pass job/checkpoint_path.

    Per-segment loop:
    1. Compute filename: f"seg_{seg.index:04d}.wav"
    2. Compute path: os.path.join(output_dir, "segments", filename)
    3. If resume=True and seg.status == SegmentStatus.TTS_DONE: print skip message, continue
    4. If not should_synthesize(seg.ru_text):
       - Write silence WAV (see below)
       - seg.audio_path = path
       - seg.status = SegmentStatus.SKIPPED
       - Print: f"[{seg.index:02d}/{total}] {filename} — SKIPPED (silence)"
    5. Else: call _call_with_retry(client, seg.ru_text, voice_id, ...) → pcm_bytes
       - Wrap pcm_bytes in WAV container via _write_pcm_as_wav(pcm_bytes, path)
       - seg.audio_path = path
       - seg.status = SegmentStatus.TTS_DONE
       - Print: f"[{seg.index:02d}/{total}] {filename} — OK"
       - On exhaustion (raises after 5 attempts): catch exception, mark ERROR, print error line
    6. Call checkpoint.save(job, checkpoint_path) if job and checkpoint_path are not None

    BACKOFF_SCHEDULE = [5, 10, 20, 40, 80]  (5 entries = max 5 attempts)

    _call_with_retry(client, text, voice_id, model_id, stability, similarity_boost) -> bytes:
    - for attempt, wait_default in enumerate(BACKOFF_SCHEDULE):
        try: return _synthesize_segment(...)
        except ApiError as exc:
          if attempt < len(BACKOFF_SCHEDULE) - 1:
            if exc.status_code == 429:
              retry_after = exc.headers.get("Retry-After") if exc.headers else None
              wait = float(retry_after) if retry_after else wait_default
            else:
              wait = wait_default
            print(f"... attempt {attempt+2}/{len(BACKOFF_SCHEDULE)}, waiting {wait:.0f}s...")
            time.sleep(wait)
          else:
            raise  # propagate after last attempt

    _synthesize_segment(client, text, voice_id, model_id, stability, similarity_boost) -> bytes:
    - audio_iter = client.text_to_speech.convert(
          text=text, voice_id=voice_id, model_id=model_id,
          output_format="pcm_22050",
          voice_settings=VoiceSettings(stability=stability, similarity_boost=similarity_boost),
      )
    - return b"".join(audio_iter)  # MUST collect iterator

    _write_pcm_as_wav(pcm_bytes: bytes, path: str) -> None:
    - wave.open(path, "wb") as wf:
        wf.setnchannels(1); wf.setsampwidth(2); wf.setframerate(22050)
        wf.writeframes(pcm_bytes)

    _write_silence(path: str, duration_sec: float) -> None:
    - sample_rate = 22050
    - n_frames = int(sample_rate * duration_sec)
    - silence = b"\x00" * (n_frames * 2)  # 1 channel × 2 bytes (16-bit S16LE)
    - wave.open(path, "wb") as wf:
        wf.setnchannels(1); wf.setsampwidth(2); wf.setframerate(22050)
        wf.writeframes(silence)

    PITFALL (Pitfall 5): Retry-After may be absent — always use .get() with fallback to schedule.
    PITFALL (Pitfall 7): os.makedirs() BEFORE any wave.open() write calls.
